==============
Code Structure
==============

Ciw is structured in an object oriented manner.


.. image:: ../_static/codestructure.svg
   :scale: 100 %
   :alt: Code structure for Ciw.
   :align: center

Ciw consists of 3 types of objects, Core, Input, and Optional.

Core:

- Simulation
- Arrival Node
- Exit Node
- Node
- Server
- Individual
- Data Record

Input:

- Network
- Service Centre
- Customer Classe

Optional:

- State Tracker
- Deadlock Detector
